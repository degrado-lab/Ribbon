Ribbon

Ribbon is a python package which simplifies the usage and pipelining of biological software. Installing and running state-of-the-art tools can be done with a simple python script.

Quick Start

Ready to dive in? Follow these simple steps:

  1. Installation:
    Get started by following our Installation Guide which details how to set up Ribbon on your system.

  2. Getting Started:
    Explore practical examples in our Usage Guide that demonstrate common workflows.

Please visit our GitHub repo for the Ribbon source code. Ribbon is open-source, and MIT licensed.
